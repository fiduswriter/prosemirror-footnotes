Issues

  1. It is possible to copy a footnote into the footnote editor. Why? The schema shouldn't allow that.
  2. It is possible to destroy the containers that hold the footnotes in the footnote editor. It would be good if these could be locked.
  3. If copying a piece of a text which includes a footnote, and then inserting that text somewhere later in the document, the footnote appears a second time (that's good). But if editing the second footnote, the first is changed. This must be because the same footnote Node object is in the document twice after editing. Is this how it should be?
